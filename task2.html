Task 2: Get the books based on ISBN

API Endpoint

GET /api/books/{isbn}

This endpoint returns details for a specific book based on its ISBN.

Example Request

fetch('/api/books/9780743273565')
.then(response => response.json())
.then(data => console.log(data));
