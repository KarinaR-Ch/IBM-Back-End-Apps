Task 5: Get Book Reviews

API Endpoint

GET /api/books/{isbn}/reviews

This endpoint returns all reviews for a specific book based on its ISBN.

Example Request

fetch('/api/books/9780743273565/reviews')
.then(response => response.json())
.then(data => console.log(data));
