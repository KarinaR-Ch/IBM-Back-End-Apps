Task 9: Delete Book Review

API Endpoint

DELETE /api/books/{isbn}/reviews/{reviewId}

This endpoint deletes a specific review for a book.

Example Request

fetch('/api/books/9780743273565/reviews/rev_12345', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer {token}'
  }
})
.then(response => response.json())
.then(data => console.log(data));

API Requirements

Authentication: Requires valid JWT token

Permissions: Users can only delete their own reviews

Response: Returns success message upon deletion

Instructions

  1. Select a review from your existing reviews
  2. Click the "Delete Review" button
  3. Confirm deletion in the confirmation dialog
  4. View the API response
  5. Take a screenshot for submission
